# Название проекта: Catch the Balls (Поймай шарики).

Это аркадная игра на Java Swing. Игрок управляет корзинкой с помощью мыши и должен ловить зелёные шарики, которые дают очки, и избегать красных, которые отнимают жизнь.
При ловле зелёных шариков растёт комбо, увеличивающее количество очков. Игра заканчивается, когда
заканчиваются жизни (по умолчанию 3) или когда игрок набирает 60 очков (победа).

Управление:
- Мышь: движение корзинки влево-вправо.
- N: начать новую игру.
- P: поставить игру на паузу / снять с паузы.
- Q: выйти из игры.
